# 🚀 Streamlit Portfolio

Selamat datang di repositori portofolio saya yang dibangun dengan **Streamlit**.  
Repositori ini berisi kumpulan proyek data interaktif, visualisasi, dan aplikasi machine learning.

## 📊 Fitur Utama

- **Dashboard Interaktif** – Visualisasi data real-time dengan filter dinamis.
- **Machine Learning Demo** – Prediksi dan eksplorasi model secara langsung.
- **Analisis Eksploratif (EDA)** – Ringkasan statistik dan plot otomatis.
- **Upload & Proses CSV** – Pengguna dapat mengunggah file CSV untuk dianalisis.
- **Desain Responsif** – Tampilan optimal di desktop maupun mobile.

## 🛠️ Teknologi yang Digunakan

- [Streamlit](https://streamlit.io/) – Framework utama untuk UI dan interaktivitas.
- Pandas, NumPy – Manipulasi data.
- Plotly, Matplotlib, Seaborn – Visualisasi.
- Scikit-learn – Model machine learning sederhana.
- Python 3.10+
